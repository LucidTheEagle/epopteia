# Contributing to Epopteia

Personal brand project. Feedback and issue reports welcome.

---

## Philosophy

- Sharp edges — zero rounded corners
- Crystalline Silver #C0C0C0 accent — never cyan
- PREMIUM_EASE [0.16, 1, 0.3, 1] — intentional motion only
- Direct declarative voice

## Commit Format
```
feat: add component
fix: correct behaviour
style: refine visual
docs: update docs
refactor: simplify logic
chore: update deps
```

## What Will Not Be Accepted

- Rounded corners
- Cyan (#00f0ff) — replaced by Crystalline Silver
- Decorative animations
- Soft or generic SaaS aesthetics
